# ⚙️ Micro-Monitor de Rendimiento para PC (Estilo Minimalista)

Un widget flotante y compacto diseñado en Python para supervisar los recursos del sistema en tiempo real sin consumir espacio en pantalla ni rendimiento.

##  Características
- **Diseño Compacto:** Ventana flotante estilo micro-widget (260x220px).
- **Modo Siempre Visible:** Se mantiene por encima de juegos y aplicaciones (`topmost`).
- **Estilo Cyberpunk:** Interfaz oscura con barras de progreso delgadas en colores neón.
- **Monitoreo en Tiempo Real:** Actualización fluida cada 500ms de CPU, RAM y Disco D.

## Tecnologías Utilizadas
- **Python 3**
- **CustomTkinter** (Para la interfaz gráfica moderna)
- **Psutil** (Para la lectura de componentes del sistema de Windows)
- **Pillow** (Gestión de librerías visuales)

## Instalación y Uso

1. Clona o descarga este proyecto.
2. Instala las dependencias necesarias en tu terminal:
   ```bash
   pip install customtkinter psutil

# PC Performance Micro-Monitor (Minimalist Style)

A floating and compact widget designed in Python to monitor system resources in real-time without consuming screen space or performance.

## Features
- **Compact Design:** Floating micro-widget style window (260x220px).
- **Always-on-Top Mode:** Stays above games and applications (`topmost`).
- **Cyberpunk Style:** Dark interface with thin progress bars in neon colors.
- **Real-Time Monitoring:** Smooth 500ms updates of CPU, RAM, and Drive D.

## Technologies Used
- **Python 3**
- **CustomTkinter** (For the modern graphical interface)
- **Psutil** (For reading Windows system components)
- **Pillow** (Visual library management)

## Installation and Usage

1. Clone or download this repository.
2. Install the required dependencies in your terminal:
   ```bash
   pip install customtkinter psutil
   ```
3. Run the program:
   ```bash
   python monitor_pc.py
   ```
